belz web               — Manage the Belzabar web app (Next.js, port 65535)
  belz web             — Show status + available subcommands (does not start)
  belz web start       — Start the web app in the background (-v for foreground)
  belz web stop        — Stop the running web app
  belz web restart     — Restart the web app
  belz web status      — Check running + autostart state
  belz web enable      — Start the web app automatically at login
  belz web disable     — Remove the login autostart
