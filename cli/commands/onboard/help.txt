belz onboard

Guided first-run setup for belz. Walks through, in order:

  1. Credentials  — environment usernames/passwords (via `belz setup`)
  2. Extension    — force-install the browser extension (via `belz extension load`)
  3. Web app      — start the web app automatically at login (via `belz web enable`)

Each step is confirmed before it runs and is safe to skip. `belz onboard` is
idempotent — re-run it any time; steps that are already done are detected and
offered for skipping.

This command is interactive. For non-interactive / scripted setup, run the
underlying commands directly:
  belz setup --env-file <path>
  belz extension load --all
  belz web enable

Examples:
  belz onboard            Run the guided setup
