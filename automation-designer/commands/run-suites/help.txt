Usage: belz ad run-suites [flags]

Execute all saved regression test suites found in the 'suites/' directory.
Each suite is verified for HTTP 200 and a non-failed execution result.

Flags:
  --help, -h          Show this help message.

Global Flags:
  --env <name>        Set active environment. (default: nsm-dev)
                      Available: nsm-dev | nsm-qa | nsm-uat
  --llm               Output structured JSON envelope for scripting/LLM use.

Examples:
  belz ad run-suites
  belz ad run-suites --env nsm-qa
  belz ad run-suites --llm
