Unified CLI

Native binary built with Bun. Inspect methods, run tests with trace logs, and trigger migrations. Supports `ad`, `pd`, and `migrate` namespaces out of the box.

Smart Extension

Manifest V3 extension for Chrome. JSON bulk editors, CodeMirror modals, and keyboard shortcuts for the NSM web platform.

Zero Auth Latency

Automatic session persistence and 401 refresh logic. Login once, work across all NSM environments without manual token management.

AI Ready (ACP)

Full support for the Agent Client Protocol. Use Belz AI as a bridge for coding agents like Claude or Gemini to interact with your local environment.

Data Flow

Tracing a standard authenticated request from CLI to NSM.

1. CLI Dispatch

Runner identifies the namespace (e.g., ad) and locates the command in the generated registry.

2. Auth Reconcile

Core auth.ts checks local session. If missing or expired, it re-logs in using credentials from config.json.

3. API Invocation

Request is sent via apiFetch. If a 401 occurs, the session is cleared and the request is retried once after re-auth.

4. Hydration & Display

Raw JSON is parsed into domain types. renderHuman (tables) or renderLLM (JSON) prints to stdout.

Belz AI Web App

The modern UI for connecting local coding agents to your browser.

The ACP Bridge

Browsers cannot spawn processes. Belz AI uses a Node.js singleton bridge that proxies ACP JSON-RPC requests to agent subprocesses via stdio.

// lib/acp-bridge.ts class AcpBridge { private sessions = new Map(); async sendPrompt(id: string, prompt: string, emit: Emitter) { const entry = this.sessions.get(id); return entry.connection.prompt({ sessionId: entry.acpId, prompt }); } }

State Management

Uses a centralized useReducer hook to manage multi-session state, ensuring SSE stream updates don't cause race conditions or stale closures.
