Usage: belz pd analyze [PAGE_ID] [flags]

Recursively analyze a page, collecting all nested components and AD method references.
Uses the default page list from config when PAGE_ID is omitted.

Arguments:
  [PAGE_ID]           UUID of the page to analyze. Uses default pages if omitted.

Flags:
  --compliance        Verify discovered AD IDs against master_ids.txt.
  --help, -h          Show this help message.

Global Flags:
  --env <name>        Set active environment. (default: nsm-dev)
                      Available: nsm-dev | nsm-qa | nsm-uat
  --llm               Output structured JSON envelope for scripting/LLM use.

Examples:
  belz pd analyze
  belz pd analyze 49fa2576...
  belz pd analyze 49fa2576... --compliance
  belz pd analyze --compliance --llm
