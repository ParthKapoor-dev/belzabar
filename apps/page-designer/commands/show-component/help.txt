Usage: belz pd show-component <NAME> [flags]

Search for a Page Designer component by name and display its configuration.

Arguments:
  <NAME>              The name of the component to show.

Flags:
  --full              Include the full configuration payload.
  --raw               Include the raw configuration string in output.
  --help, -h          Show this help message.

Global Flags:
  --env <name>        Set active environment. (default: nsm-dev)
                      Available: nsm-dev | nsm-qa | nsm-uat
  --llm               Output structured JSON envelope for scripting/LLM use.

Examples:
  belz pd show-component n_s_staff_email_correspondence
  belz pd show-component n_s_staff_email_correspondence --full
  belz pd show-component n_s_staff_email_correspondence --raw --llm
