Usage: pd show-page <PAGE_ID> [flags]

Fetches and displays the configuration of a specific Page Designer page.

Arguments:
  <PAGE_ID>     The UUID of the page to show.

Flags:
  --full        Include full configuration payload.
  --raw         Include raw configuration string in output.
  --help, -h    Show this help message.

Example:
  pd show-page 49fa2576...
