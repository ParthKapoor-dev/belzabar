Usage: belz pd find-ad-methods <ID> [flags]

List all Automation Designer method IDs referenced in a page or component.

Arguments:
  <ID>                UUID of the page, or name of the component.

Flags:
  --component         Treat <ID> as a component name instead of a page UUID.
  --recursive, -r     Deep-scan all child components recursively.
  --help, -h          Show this help message.

Global Flags:
  --env <name>        Set active environment. (default: nsm-dev)
                      Available: nsm-dev | nsm-qa | nsm-uat
  --llm               Output structured JSON envelope for scripting/LLM use.

Examples:
  belz pd find-ad-methods 49fa2576...
  belz pd find-ad-methods n_s_staff_email_correspondence --component
  belz pd find-ad-methods 49fa2576... --recursive --llm
