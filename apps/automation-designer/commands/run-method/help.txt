Usage: belz run-method <PUBLISHED_ID> [PAYLOAD] [flags]

Executes an Automation Method.

Arguments:
  <PUBLISHED_ID>   The unique ID of the PUBLISHED method version.
  [PAYLOAD]        Optional. JSON string OR path to a JSON file. 
                   Defaults to '{}' if omitted.

Flags:
  --raw            Include raw payload in output data.
  --help, -h       Show this help message.

Examples:
  belz run-method 40ba5566efa95b033ef7af8f7f30ee09 '{"type":"LT262A"}'
  belz run-method 40ba5566efa95b033ef7af8f7f30ee09 ./data/payload.json
  belz run-method 40ba5566efa95b033ef7af8f7f30ee09 ./data/payload.json --raw --llm
