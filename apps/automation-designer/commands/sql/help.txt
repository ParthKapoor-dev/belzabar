Usage: belz sql <subcommand> [args] [flags]

Run SQL read-mode queries via Automation Designer database services.

Subcommands:
  run <query>        Execute a SQL query in read mode.
  dbs                List available database auth configurations.

Flags (run):
  --db <name|id>     Optional database nickname or id.
                     Default resolution order:
                     1) BELZ_SQL_DEFAULT_DB
                     2) NSM_Read_DB
  --raw              Include raw operation metadata, payload, and execution response.

Flags (dbs):
  --raw              Include raw API response.

Examples:
  belz sql dbs
  belz sql run "select * from users limit 1"
  belz sql run "select * from users limit 1" --db NSM_QA_DB
  BELZ_SQL_DEFAULT_DB=NSM_QA_DB belz sql run "select now()"
  belz sql run "select * from users limit 1" --raw --llm
